# Парсер Wildberries

Асинхронный парсер товаров с маркетплейса Wildberries. 
Разработан в учебных целях для демонстрации архитектуры, паттернов проектирования и качества кода на Python.

## Стек технологий
- Python 3.10+
- `aiohttp` — асинхронные HTTP-запросы
- `pydantic` — строгая типизация и валидация
- `loguru` — красивое и удобное логирование
- `pytest` — тестирование

## Установка
```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```
